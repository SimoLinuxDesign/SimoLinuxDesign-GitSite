INSTALLATION

Vediamo come poter installare Xampp sul vostro dispositivo linux. Per prima cosa, dobbiamo essere a conoscenza che verranno installate diverse librerie e verranno scaricati dei file da internet esterni a questo sito in quanto questo è solo uno script che ne facilità l'installazione e l'utilizzo.


L'installazione può essere effettuata sia in modo "Manuale che Automatica". Solitamente consiglio l'utilizzo dell'installazione automatica se si hanno le versioni di linux indicate nelle compatibilità di questa pagina.


Spiegherò entrambi i metodi di installazione in quanto i processi di installazione verranno divisi in diversi steps.

Step 1: [Prepariamo il file d'installazione.]

Per prima cosa rechiamoci sul repository git in modo da poter scaricare il programma di installazione (da qui;). Scarichiamo la cartella sul desktop (solitamente: /home/nomeutente/Scrivania) se possibile, altrimenti, una volta finito il download spostiamo la cartella appena scaricata da "Scaricati" (o Download) a Scrivania (o Desktop)

Ora apriamo il terminale e spostiamoci nella cartella scaricata con il comando "cd"

cd /home/nomeutente/Scrivania/xampp

Sostituire "nomeutente" con "il l'user

Una volta entrati nella cartella, diamo i permessi di lettura, esecuzione e scrittura dello script nominato "install.sh" con il comando "chmod 777" con i permessi di super-utente, così da poter proseguire con l'installazione.

sudo chmod 775 install.sh
P.S. Per poter funzionare correttamente, sudo deve essere installato e configurato.

Step 2: [Avviamo l'installazione automatica]

Sempre dal terminale, eseguiamo il comando:

sudo ./install.sh

Inseriamo la password (che potrà essere richiesta più volte durante questo processo).

sudo chmod 775 install.sh

Una volta avviato, ci verrà visualizzata la descrizione dello script, e sopratutto delle informazioni essenziali per la configurazione quali sono "Sistema Operativo" e "Desktop Environment", fondamentali per un'installazione manuale. Una volta annotate le informazioni premiamo "Invio" per proseguire con l'installazione.

Il programma controllerà che xampp non sia già installato sul sistema operativo o che xampp quanto meno non sia già stato scaricato.
